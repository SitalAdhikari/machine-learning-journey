## Day 27 – PCA Theory (Deep Dive)

Today I continued learning the **theory behind PCA**.

### What I Learned

* PCA finds directions with **maximum variance**
* **Principal Components** are new uncorrelated features
* **Eigenvectors** → Direction of components
* **Eigenvalues** → Importance of components

### Key Insight

PCA reduces dimensions while keeping the **most important information** in the dataset.

### Takeaway

Understanding PCA mathematically helps in better **feature extraction and dimensionality reduction**.
