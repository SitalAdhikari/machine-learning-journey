# Day 35 — Polynomial Linear Regression

## 📌 Overview

On **Day 35**, I learned **Polynomial Linear Regression**, which helps model **non-linear relationships** using linear regression by adding polynomial features.

## 🧠 Why Polynomial Regression?

Simple Linear Regression works only for **linear data**, but real-world data is often **curved (non-linear)**.
Polynomial Regression solves this by transforming features.

## 📈 Polynomial Equation

[
y = b_0 + b_1x + b_2x^2 + b_3x^3 + ... + b_nx^n
]

## 🚀 What I Learned

* Non-linear relationships
* Polynomial feature transformation
* Model fitting with polynomial degree
* Underfitting vs Overfitting concept
