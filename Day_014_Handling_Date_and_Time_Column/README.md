## Day 14 – Handling Date & Time Columns

Today I learned how to handle **Date and Time features in Machine Learning datasets**.
Date columns cannot be directly used by models, so we need to **extract meaningful numerical features** from them.

### What I Learned

* Converting columns to **datetime format**
* Extracting useful features such as:

  * **Year**
  * **Month**
  * **Day**
  * **Day of Week**
* Using these extracted features for **model training**

### Why It Matters

Date and time often contain **hidden patterns** (seasonality, trends, weekly behavior).
By converting them into numerical components, machine learning models can **capture temporal patterns** effectively.
