📈 Softmax Loss Formula

Loss = -\sum y_i \log(p_i)

Where:

 = Actual class

 = Predicted probability


💡 Key Takeaway

Softmax Loss measures how well the predicted probabilities match the actual class labels.

Lower Loss → Better Model Performance
