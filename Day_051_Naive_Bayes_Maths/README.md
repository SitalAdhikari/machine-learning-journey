Day 51 — Naive Bayes (Mathematical Foundation)

📌 Overview

On Day 51, I studied the mathematics behind Naive Bayes, a probabilistic algorithm used for classification.

🧠 What I Learned

Conditional Probability

Bayes Theorem

Likelihood, Prior, Posterior

Independence assumption in Naive Bayes


📈 Bayes Theorem

P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}

🤖 Naive Bayes Idea

P(Class | Features) \propto P(Class) \cdot P(Features | Class)

Assumes features are independent

Computes probability for each class

Chooses class with highest probability


💡 Key Takeaway

Naive Bayes is simple yet powerful — it uses probability and independence assumption for classification.
