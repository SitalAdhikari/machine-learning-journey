## Day 28 – PCA on MNIST Dataset

Today I applied **Principal Component Analysis (PCA)** on the **MNIST dataset** to reduce dimensionality.

### What I Did

* Loaded the MNIST handwritten digit dataset
* Scaled the features
* Applied PCA for dimensionality reduction
* Reduced high-dimensional image data into fewer components
* Observed explained variance

### Why PCA on MNIST?

MNIST images have **many features (pixels)**.
PCA helps:

* Reduce dimensions
* Speed up training
* Remove noise
* Keep important information

### Key Insight

Even after reducing dimensions, PCA still preserves **most of the useful information** in image data.

### Takeaway

PCA is very powerful for **high-dimensional datasets like images**, making models faster and more efficient.
