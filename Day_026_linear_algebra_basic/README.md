## Day 26 – Linear Algebra Basics for Machine Learning

Today I started learning the **fundamentals of Linear Algebra**, which is the backbone of many machine learning algorithms.

---

### What I Learned

* **Vectors**: Represent data points in space

* **Matrices**: Represent datasets and transformations

* **Matrix Operations**:

  * Addition
  * Multiplication
  * Transpose

* **Dot Product**:

  * Measures similarity between vectors
  * Widely used in ML algorithms

---

### Why It Matters

* Machine learning models internally work with **vectors and matrices**
* Concepts like **PCA, Linear Regression, and Neural Networks** rely heavily on linear algebra
* Helps in understanding how algorithms work mathematically

---

### Key Insight

Linear Algebra is not just theory — it is the **language of machine learning**.

---

### Final Takeaway

A strong foundation in linear algebra is essential to move from **using ML models → understanding ML models deeply**.
