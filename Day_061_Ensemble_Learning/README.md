## Day 61: Introduction to Ensemble Learning
Today, I started exploring **Ensemble Learning**, a powerful technique based on the "Wisdom of the Crowd".
### Key Learnings
 * **Core Concept**: Combining multiple machine learning models to create a more robust "super-model" that reduces noise, bias, and variance.
 * **Diversity is Key**: To be effective, the individual models (M_1, M_2, ... M_n) should be different—either by using different algorithms, different data subsets, or a combination of both.
 * **Aggregation Logic**:
   * **Classification**: Uses a majority vote to determine the final class.
   * **Regression**: Takes the average (mean) of all model predictions.
 * **Trade-off**: While ensemble methods generally improve performance, they increase computational complexity.
