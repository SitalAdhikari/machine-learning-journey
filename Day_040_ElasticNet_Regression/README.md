Day 40 — Elastic Net Regression

📌 Overview

On Day 40, I learned Elastic Net Regression, which combines Ridge (L2) and Lasso (L1) regularization.

🧠 Why Elastic Net?

Ridge reduces overfitting

Lasso performs feature selection

Elastic Net combines both advantages


📈 Formula

Loss = RSS + \lambda_1 \sum |w| + \lambda_2 \sum w^2

L1 (Lasso) → Feature selection

L2 (Ridge) → Reduce overfitting


💡 Key Takeaway

Elastic Net provides balanced regularization and works well when features are highly correlated.
