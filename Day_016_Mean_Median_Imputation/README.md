## Day 16 – Mean & Median Imputation

Today I learned how to handle **missing numerical data using Mean and Median Imputation**.

### Key Concepts

* **Mean Imputation**: Replace missing values with the average of the column.
* **Median Imputation**: Replace missing values with the middle value of the column.

### What I Observed

* Imputation can **change variance and distribution** of the data.
* Sometimes the distribution changes a lot (bad sign), and sometimes it stays similar (good sign).
* Median is usually better when **outliers are present**.

### Key Takeaway

Mean/Median imputation is simple and useful, but we must **check how it affects the data distribution before using it in models**.
