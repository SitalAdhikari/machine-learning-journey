## Day 80: K-Means Clustering
Today, I explored **K-Means Clustering**, an unsupervised learning algorithm used for data partitioning.
### Key Learnings
 * **Core Pipeline**:
   1. Define the number of clusters (K).
   2. Initialize random cluster centroids.
   3. Assign points using Euclidean distance.
   4. Recalculate centroids iteratively until convergence.
 * **Elbow Method**: Used to determine the optimal K by plotting the Within-Cluster Sum of Squares (WCSS) against the cluster count and finding the "elbow point".
