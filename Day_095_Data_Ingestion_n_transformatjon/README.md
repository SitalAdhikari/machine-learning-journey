## Data Ingestion: Bringing It In
Data ingestion is the process of **transporting data** from various sources (like databases, IoT devices, or apps) into a centralized storage system (like a data lake or warehouse). Think of it as scooping up the raw data and getting it through the door, either in real-time (streaming) or in chunks (batch).
## Data Transformation: Cleaning It Up
Data transformation is the process of **converting and shaping** that raw data into a usable format. Since raw data is often messy, this step involves cleaning, filtering, normalizing, and structuring it so that analysts and AI tools can actually make sense of it.
