Day 52 — K-Nearest Neighbors (KNN)

📌 Overview

On Day 52, I learned K-Nearest Neighbors (KNN), a simple and intuitive algorithm used for classification and regression.

🧠 What I Learned

Instance-based learning

Distance metrics (Euclidean Distance)

Choosing value of K

Majority voting (for classification)


📈 How KNN Works

1. Choose value of K


2. Calculate distance from query point to all data points


3. Select K nearest neighbors


4. Predict based on majority class (or average for regression)



📏 Distance Formula (Euclidean)

d = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}

💡 Key Takeaway

KNN makes predictions based on similarity (distance) between data points.
