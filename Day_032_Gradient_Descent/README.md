Day 32 — Gradient Descent

Implemented Gradient Descent to optimize model parameters by minimizing the cost function.

🧠 Formula

\theta = \theta - \alpha \cdot \frac{dJ}{d\theta}

θ → Parameters

α → Learning rate

dJ/dθ → Gradient
