## Day 58: Entropy and Information Theory
Today I focused on **Entropy**, the mathematical measure of impurity or "disorder" within a dataset.
### Key Learnings
 * **Core Formula**: H(S) = \sum -p_i \log_2(p_i), where p_i is the probability of class i.
 * **Entropy Scale**: For binary classes, entropy ranges from **0** (perfectly pure) to **1** (maximum uncertainty).
 * **Predictability**: High entropy indicates high uncertainty, making predictions more difficult.
 * **Multi-class**: While the minimum remains 0, entropy can exceed 1 for datasets with more than two classes.
Understanding this "chaos" is essential for deciding how Decision Trees split data effectively!
