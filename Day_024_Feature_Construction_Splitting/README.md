## Day 24 – Feature Construction & Splitting

Today I learned about **Feature Construction and Feature Splitting**, two important techniques in feature engineering.

---

### 1. Feature Construction

Feature construction means creating **new features from existing ones** to improve model performance.

Examples:

* Combining features (e.g., `FamilySize = SibSp + Parch`)
* Creating meaningful ratios or interactions
* Deriving new insights from existing data

🔹 Helps models capture **hidden patterns**
🔹 Improves predictive power

---

### 2. Feature Splitting

Feature splitting means breaking a single feature into **multiple meaningful parts**.

Examples:

* Splitting **full name → first name + last name**
* Extracting **title (Mr, Mrs, etc.)**
* Separating combined or complex columns

🔹 Makes data more structured
🔹 Improves model interpretability

---

### What I Did

* Created new features from existing columns
* Split complex features into simpler components
* Analyzed how these transformations improve data understanding
